# 🧪 Rick and Morty Explorer

¡Bienvenido al explorador definitivo del multiverso! Esta es una Single Page Application (SPA) de alto rendimiento construida con **React** y **Vite**, que consume la API oficial de Rick and Morty para ofrecer una experiencia de usuario fluida, robusta y totalmente responsiva.

---

## ✨ Características Principales

- **🔍 Búsqueda Inteligente:** Sistema de filtrado por nombre con **Debounce** (500ms) para optimizar las peticiones a la API y reducir el tráfico innecesario.
- **🎭 Filtros por Estado:** Clasificación instantánea de personajes (Alive, Dead, Unknown) con feedback visual inmediato.
- **📑 Paginación Dinámica:** Implementación de "Cargar más" con gestión de estados inmutables para una exploración sin interrupciones.
- **❤️ Sistema de Favoritos:** Persistencia de datos mediante **LocalStorage**, garantizando que tus personajes elegidos se mantengan guardados localmente incluso tras recargar la página.
- **📱 Diseño Mobile-First:** Interfaz 100% responsiva utilizando **CSS Grid** y **Flexbox** con breakpoints específicos para móviles, tablets y escritorio.
- **🛡️ Resiliencia y Errores:** Manejo global de excepciones de red y estados de carga con lógica de reintento (Retry) para una experiencia de usuario profesional.
- **📦 Detalle en Modales:** Portales de información extendida para cada personaje, incluyendo origen, especie y lista de episodios.
